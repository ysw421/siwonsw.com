# MDX 통합 예제

이것은 **MDX** 파일의 예제입니다. Markdown과 React 컴포넌트를 함께 사용할 수 있습니다.

기술의 세계에서 0부터 무한대까지의 여정은 끝이 없습니다. 우리는 항상 배우고, 탐구하고, 혁신합니다. 새로운 기술, 새로운 아이디어, 새로운 가능성을 향한 우리의 여정은 계속될 것입니다. 0-inf의 비전은 간단합니다. 끊임없이 배우고, 성장하며, 혁신하는 것. 0부터 시작해 무한대를 향해 나아가는 우리의 여정은 앞으로도 계속될 것입니다.
